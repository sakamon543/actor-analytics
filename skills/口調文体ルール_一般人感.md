# 口調・文体ルール（一般人感／プレイヤー感）

このファイルは、AI感を消して**実際のSNSプレイヤーが書いてる感**を出すためのルール集。
CSV50本（復縁みるく・もも・R・凪・ななせまる・てぃあ・みいこ等）から抽出。

スキル本体の文体チェックで参照する。

---

## 大原則

**AI感の正体＝「形式的に整いすぎてる」「抽象語が多い」「比喩がビジネスっぽい」**

**一般人感／プレイヤー感の正体＝「省略形」「口語」「具体名詞」「文の長さがバラバラ」「感情語」**

## NG単語・表現リスト（使うとAI感が出る）

### AI造語・「使わない言葉」禁止リスト ★最優先★

**判定基準：友達に話す時／普段のLINEで使う？ Noなら NG**

| AI造語パターン | 例 | 自然な置き換え |
|---|---|---|
| 〇〇状態 | フリーズ状態、処理状態、固まってる状態 | 動けない／何していいか分からない／迷ってる |
| 〇〇する側 | 気持ちある側、振った側、行動する側 | 気持ちある方／振った方（「方」の方が自然） |
| 〇〇を解除する／〇〇を解く | フリーズを解除する／固まりを解く | 動き出させる／返しやすくする |
| 〇〇期に入る | 美化期、揺らぎ期、整理期 | 〇〇する時期 |
| 〇〇モード | 防衛モード、回避モード | 〇〇な時／〇〇してる時 |
| 〇〇の処理 | 過去のこととして処理を始める | 過去のこととして整理し始める |
| 固まる／固まってる（多用） | 彼が固まってる | 動けない／迷ってる／どうしていいか分からない |
| ラベル造語 | フリーズ／揺らぎタイマー／比較タイマー | 普通の言葉で説明 |

**「日常会話チェック」：**
書いた後、各単語に対して「友達と話す時にこの言葉使う？」と自問。
使わないなら、普通の言葉に置き換え。

## OK単語・表現リスト（使うと一般人感が出る）

### 省略形（必ず使う）

| 標準形 | 省略形（推奨） |
|---|---|
| している | してる |
| していた | してた |
| してしまう | しちゃう／しちまう |
| なのですか | なの？ |
| なのではないか | なんじゃない？ |
| わからない | 分からない／わからん |

### 文末の語尾（キャラに応じて使い分け）

CSV全件分析の結果、各キャラ独自の語尾傾向が判明。これをコピーする。

**経験者女性キャラ（みるく・もも・R・てぃあ・みいこ）：**
- 〜だよ／〜なんだよね（最頻出）
- 〜だからね／〜よね
- 〜らしい
- 〜なの／〜なのよ
- 〜じゃん（砕け）
- 〜だね
- 〜だったりする（R特有）
- 絵文字：🥲（最多）／😭／😌／🥺

**男性当事者「だよね」型（凪・シン・もも風男版）：**
- 〜なんですよね（最頻出）
- 〜だったりする
- 〜なんだ／〜なんだよね
- 〜って思う

**男性当事者「ですます」型（はる・N）：**
- 〜です／〜ですよ
- 〜なんです／〜なんですよね
- 〜だから／〜なので
- 「ぶっちゃけ」OK

**男性当事者「だな」型（ななせまる）：**
- 〜だな／〜なんだよな（特徴的）
- 〜と思う／〜と思って
- 〜るぞ（断言）

**専門家系（ハクオウ等）：**
- 〜だ／〜じゃない（短文断言）
- 〜と思う（柔らかい）
- 〜してる／〜できてる（省略形）
- 〜のせい／〜だけ（限定）
- 〜って（伝聞）
- 「〜だよね」は**控えめ**（フランクすぎ）
- 絵文字は**基本不使用**

### 接続詞・つなぎ言葉

| 推奨 | 効果 |
|---|---|
| 〜から | 「〜なので」より自然 |
| 〜けど／〜けどさ | 「〜しかし／〜けれども」より自然 |
| 〜って／〜って思って | 「と思って」より口語 |
| 〜なのに | 矛盾を強調 |
| とか／とかとか | 列挙の口語 |
| ぶっちゃけ／正直 | カジュアル開示（キャラによる） |
| ちょっと | 柔らかい強調 |

### 言いさし表現（追加：CSV全件分析で発見）

文末を完全にしない／ニュアンスを残す表現：

- 〜だったりする（R・凪が頻用、リアル感を一気に上げる）
- 〜まうんだよね／〜しちゃう／〜してた
- 〜なのよ／〜なんですよね
- 〜だな／〜なんだよな（ななせまる風）
- 〜だな／〜だよ／〜だね（経験者女性）
- 〜らしい（伝聞）
- 〜って／〜って思う／〜って気づいた

### 強調副詞・口語強調

| 推奨 | 用法 |

### 「」（カギ括弧）の使い方

CSVでは「」を**頻繁に**使ってる。AI生成物より圧倒的に多い：

**用途1：男のセリフ／本音**
> 「自分から振ったのに戻りたいなんて都合よすぎる」
> 「今さら連絡したら迷惑だろう」

**用途2：女性の心の声／読者の声**
> 「もう私のこと忘れたんだ」って絶望してる
> 「私のどこがダメだったの？」って自分を責めてる

**用途3：購入者／相談者の発言**
> 「冷却期間3ヶ月取ったけど何も起きません」って泣いてた
> 「最近変わったね」って言われた

**用途4：常識・通念の引用**
> 「冷却期間3ヶ月」って信じてる子
> 「自分磨きすれば振り向く」って思い込んでる

**ルール：1ポストあたり3〜6箇所が自然。**少なすぎるとリアリティ不足、多すぎるとうるさい。

**CSV全件分析の結果：平均5〜8箇所。3箇所未満のポストはAIっぽく見えるリスク高。**

## 句読点テクニック

### 句点（。）の戦略的使い方

CSVの実例：
> "気持ちはある。動けないだけ。"
> "ほとんどの男には逆効果。"
> "これ、女性には理解しがたいと思う。でも男の脳はそうできてる。"

**短文断言にこそ「。」を打つ。** 長い説明文の途中の「。」は読点（、）に置き換える。

### 読点（、）で繋ぐ

CSVの実例：
> "全国の復縁を目指す女子は諦めなくていいよ。私の彼も別れ際は「二度と会わない」ってキレ気味で言ってきたけど、今となっては彼の方が私に溺愛してる。"

→ 1文がかなり長い。「、」と「けど／から／って」で繋ぐ。「。」を頻発させない。

### 文末「。」の省略

CSVの実例：
> "彼から「最近変わったね」って言われた"
> "全部「もう恋愛感情がない男」の対応"

文末の「。」を省略してもOK。リズムが軽くなる。

### 「↓」で引きを作る

メインの最後に「↓」を置いてぶら下げに引き込む。CSV頻出パターン。

### 文の長さをバラけさせる

**NG（AI的）：**
```
彼は気持ちがある。
でも動けない。
だから連絡しない。
```
（全部同じ長さ）

**OK（自然）：**
```
彼は気持ちがある。動けないだけ。
連絡したいのに「都合よすぎる」って自分を止めてる、これが現実。
```
（短文＋長文の混在）

## 文構造のNG（AI的）vs OK（自然）

### NG：機械的な並列

```
連絡したいのに「都合よすぎる」と自分を止める
気になるのに「もう動けない」と諦める
会いたいのに「今さら何って思われる」と怖くなる
```
→ 動詞が揃いすぎ。整いすぎ。

### OK：自然な並列

```
連絡したいのに「都合よすぎる」って止まる
気になるのに「もう動けない」って思う
会いたいのに「今さら何って」って怖くなる
```
→ 「って」を入れて自然に。動詞も少し違える。

### NG：抽象名詞の連続

```
プライドと罪悪感で身動きが取れない。
```
→ 「身動きが取れない」が形式的。

### OK：具体的描写

```
プライドと罪悪感で動けない。
頭では連絡したいのに、指が止まる。
```
→ 動作が見える。

### NG：「〜的」「〜性」「〜化」の多用

```
心理的に厄介で、構造的に問題があり、本能的に動けない状態に固定化されてしまう。
```
→ 抽象名詞化（AI／論文っぽい）

### OK：動詞中心

```
心がぐちゃぐちゃで、頭で分かってても動けないまま固まる。
```

## ぶら下げの締め方パターン（CSV分析）

CTAの直前の1文。CSVで頻出するパターン：

- 〜が大事
- 〜を知ってるかどうかだけ
- 〜だけでいい
- それができれば〜
- 〜が必要
- 〜が変わる
- 〜が決まる
- 〜は意外と早い／〜は簡単に〇〇
- 〜だけで〇〇

**避ける：**
- 「〜の重要性が高まる」（堅い）
- 「〜することが本質である」（論文）
- 「〜を最大化することが鍵」（ビジネス）

## 自虐表現テンプレ（追加）

CSVで頻用される自虐パターン。3歩先ポジションを作る重要要素：

| 自虐表現 | キャラ | 効果 |
|---|---|---|
| 「私も〇〇だった」 | 経験者女性 | 共感の起点 |
| 「自分でも引くくらい」 | 経験者女性（R） | 過去の暴走を笑える形で表現 |
| 「メンヘラ全開で」 | 経験者女性（R） | 強い自虐＋親近感 |
| 「僕も同じ言葉を言ったことがある」 | 男性当事者（凪） | 当事者の証言 |
| 「本当にバカだから」 | 男性当事者（凪） | 自虐＋気づき |
| 「俺も実際それで〜」 | 男性当事者（ななせまる） | 実体験 |
| 「全部自分の中の〇〇が原因」 | 男性当事者（シン） | 自己分析 |

**専門家キャラ（ハクオウ）は自虐NG**。代わりに「僕の購入者さんで〇〇って子がいた」で他者の経験で代用。

## キャラ別調整ルール

### 経験者女性キャラ（みるく・もも・R・てぃあ・みいこ）
- 「〜だよね」「〜なんだよね」「〜らしい」多用OK
- 絵文字（🥲、😭、🥺）使用OK
- 「私も〜」「私の彼」を頻繁に出す
- 「ぶっちゃけ」「マジで」OK

### 男性当事者キャラ（凪・はる・シン・N）
- 「僕も〜」「俺も〜」を冒頭で出す
- 「ちょっと本音を言うと」「男目線で」「断言するけど」OK
- 「〜だったんだよね」「〜なんですよね」混在OK
- 自虐的な語り（「本当にバカだから」等）OK

### 専門家キャラ（ハクオウ・カウンセラー）
- 断言系（〜だ／〜じゃない）が基本
- 「〜と思う」で柔らかさ追加
- 「〜してる」「〜できてる」省略形は使う
- 「〜だよね」は控えめ（フランクすぎる）
- 絵文字は控えめか不使用
- 「臨床心理士として」「1000人見てきて」「僕の購入者さんで」を要所で使う
- 「ぶっちゃけ」「マジで」NG（権威性が落ちる）

## 一般人感を出す10のコツ

CSV分析から抽出：

1. **「。」より「、」で繋ぐ**：1文を長めに、リズムは中で作る
2. **省略形を必ず使う**：「してる」「してた」「してまう」
3. **〜って／〜けど／〜から**：3つの便利な口語接続
4. **文末の「。」を時々省略**：リズムが軽くなる
5. **具体名詞を混ぜる**：LINE／インスタ／既読／ブロック
6. **感情語を入れる**：病んでる／号泣／焦り／必死
7. **「」を多用**：男のセリフ／読者の声／購入者の言葉
8. **数字を入れる**：3週間／半年／1ヶ月／1000人
9. **文の長さをバラけさせる**：短文断言と長文説明の混在
10. **キャラの語尾を統一**：「だよね」or「だ」or「なんだ」を選ぶ

## ロング①｜本人語り_元超回避型

### メイン
```
回避型の彼に近づこうとすると避けられて、「彼にとってのちょうどいい距離が全然わからない」ってなってる人。元回避型の僕から、彼の頭の中を正直に話します。
```

### ぶら下げ
```
僕、元々は超回避型でした。当時付き合ってた6歳年上の彼女のこと、ちゃんと好きだったのに、近づかれると「重い」って感じてしまってた。

彼女は何も悪くなかったんです。ただ好きでいてくれただけ。なのに僕は返信を遅らせて、会う頻度を減らして、最後は自分から「別れよう」って言った。回避型って、嫌いだから逃げるんじゃなくて、好きだからこそ怖くて距離を取るんですよね。

だからあなたが追いかけるほど、彼は逃げる。性格が悪いんじゃなくて、回避型の脳がそう動くだけ。普通の男に効く「追わない・冷却期間」も、回避型には全然違う形で届きます。

回避型の彼が「離れたくない」に変わる距離の取り方、プロフのnoteに書いてます。
```

## ロング②(LINEランク)｜LINEランク型

### メイン
```
回避型の彼に「最近どうしてる?会いたいな」ってLINE送ろうとしてる人。それ、回避型には一番刺さらないやつです。同じ連絡でも、効くLINEは真逆なんですよね。
```

### ぶら下げ
```
回避型へのLINE、僕の感覚でランク分けするとこう。

3流：「最近どうしてる?会いたいな」← 返事を迫られてる感じがして、回避型は「重い」が発動する。
2流：「この前話してた〇〇、見つけたよ」← 用件があるぶんマシだけど、まだ"あなた発信"。
1流：「あの店行ってきた、美味しかったよ」← 返事を求めてない、ただの近況。これが効く。

回避型は「返信しなきゃ」のプレッシャーがゼロの連絡にだけ、安心して反応できるんです。会いたい・どうしてるは、彼にとって全部"宿題"に見える。

回避型の彼が思わず返したくなるLINE、タイプ別に10パターン、プロフのnoteに載せてます。
```

## ロング③(メソッド核心)｜完全に消えない_メソッド核心

### メイン
```
回避型の元彼に「もう連絡しない方がいいのかな」って完全に引こうとしてる人。ちょっと待って。完全に消えると、回避型の彼の中であなたは「過去の人」で終わります。
```

### ぶら下げ
```
僕が元カノに自分から連絡したのは、別れて4ヶ月目でした。きっかけは一個だけ。「彼女が完全には消えなかったから」です。

別れてすぐは解放感しかなかった。でも1ヶ月くらいで、スーパーで彼女が好きだった惣菜を見て足が止まったり、テレビ見て「これあいつ好きだったな」って思い出したり、じわじわ寂しくなってきた。それでも「自分から振ったくせに連絡するのはダサい」ってプライドが邪魔して動けなかった。

彼女は追いかけてこなかった。でも完全に消えてもいなかった。SNSで楽しそうにしてるのが見えて、「俺がいなくても楽しそうだな」って初めて焦った。この「ちょうどいい距離」が、回避型には一番効くんです。

追わないけど消えない「ちょうどいい距離」の作り方、プロフのnoteに書いてます。
```

## ロング④｜離れると冷たい_処理できない

### メイン
```
会ってる時は優しいのに、離れた途端に別人みたいに冷たくなる彼。「私にだけなんで?」って苦しんでる人。それ、嫌いになったんじゃなくて「処理できてない」だけなんですよね。
```

### ぶら下げ
```
元回避型の僕から言うと、会ってる時の優しさは本物です。嘘じゃない。でも離れた瞬間、回避型は「親密さ」をうまく処理できなくて、距離を取ることで自分を守ろうとする。

当時の僕も、彼女といる時は楽しいのに、家に帰ると急に「重いな」「ちょっと疲れたな」って感じて、返信が遅くなってた。彼女を嫌いになったわけじゃない。自分の中で気持ちを持て余してただけ。

ここであなたが「なんで冷たいの?」って詰めると、回避型は完全に殻に閉じこもる。逆に、彼の沈黙を責めずに「処理する時間」を渡せる人に、回避型は安心して戻ってくる。

彼の沈黙が「嫌い」じゃなく「処理中」のサインだと分かる解説、プロフのnoteに書いてます。
```

## ロング⑤｜やっちゃってる_距離置こうが3日

### メイン
```
「もう距離を置こう」って決めたのに、3日で我慢できなくて「元気?」って送っちゃう人。そして既読がつかなくて、また落ち込む。このループ、僕の元カノもやりかけてました。
```

### ぶら下げ
```
回避型相手だと、この「3日で我慢できずに送る」が一番もったいないんですよね。せっかく彼が「あれ、最近連絡こないな」って気にし始める手前で、あなたから送っちゃうと「あ、まだ好きなんだ」で彼が安心して、また距離を取る。

僕が元カノに動かされたのも、彼女が3日どころか、こっちが焦るまで連絡してこなかったから。その「間」が、回避型の僕に喪失感を生んだ。

コツは、送りたくなった衝動を一回だけやり過ごすこと。我慢じゃなくて、彼の中に「あれ?」を育てるための間だと思えば、3日が戦略に変わる。

衝動的な連絡を止める「3日ルール」の具体的な手順、プロフのnoteに書いてます。
```

## ロング⑥(商品コア)｜商品コア_自分から戻りたくなる環境

### メイン
```
回避型の彼に「追わなければ戻る」って聞いて、ただ連絡を断って待ってる人。回避型に関しては、それだけだと「興味なくなったんだ」で終わって、戻ってきません。
```

### ぶら下げ
```
元回避型の僕が断言します。回避型は「追われる」と逃げるけど、「完全に消えられる」と過去にする。どっちも復縁から遠ざかる。

効くのはその真ん中。彼の中に「喪失感」と「安心感」を同時に作ること。喪失感は「あれ、もう俺のこと好きじゃないのかな」、安心感は「でも、嫌われてはいないよな」。この2つが同時に芽生えた時、回避型は自分から動き出すんです。僕がまさにそうでした。

やることはシンプルで、追わない代わりに「ちょうどいい距離」で存在し続けるだけ。小手先の駆け引きじゃなくて、回避型の心理に沿って環境を作る。

彼が自分から「戻りたい」と思う環境の作り方、プロフのnoteに全部書いてます。
```

## ロング①(商品コア)｜商品コア_尽くすほど冷める

### メイン
```
ご飯作って、予定全部合わせて、LINEも5秒で既読つけて10回推敲して返してる人。残念だけど、その「尽くす」が彼の気持ちをどんどん遠ざけてるよ。
```

### ぶら下げ
```
これ、過去の私です。毎日ご飯作って、部屋も掃除して、彼の予定に100%合わせて。「ありがとう」も言われなくなっても、やめるのが怖くてやめられなかった。

彼の友達に「あの子、母親みたいだよな」って笑われてたのを聞いた時、息ができなくなった。私はこんなに頑張ってるのに、彼にとっては「便利な存在」でしかなかったんだって。

人って、いつでも手に入るものに価値を感じないんだよね。予定に100%合わせて、すぐ返信して、何も言わず引き下がる。全部、自分の価値を自分で下げてる行動だった。

尽くすのをやめる必要はないの。尽くし方を変えるだけ。今あなたがやってる逆効果な行動、1章のチェックリストで全部確認できるよ。プロフのnoteから。
```

```

世話を焼くのをやめろって話じゃない。量とタイミングを変えて、「一人の魅力的な女」に戻るだけ。

「お母さん」から「手放したくない女」に戻る動き方、2章に書いてるよ。プロフのnoteから。

### メイン
```

```

### ぶら下げ
```
私の相談に来た子で、毎月の記念日にプレゼント用意して、手作り料理でもてなして、彼のために何でもしてた子がいた。でもある日「一緒にいると息苦しい」って振られた。

彼の本音は「何かしてくれるたびにプレッシャーだった。返さなきゃって思うのがしんどい」だった。好きでやってるつもりでも、受け取る側は「返礼の義務」に感じてたんだよね。

尽くすこと自体は悪くない。むしろ正しい。でも「無条件に全部してあげる」は、彼に「お返しできない罪悪感」を渡してるのと同じになっちゃう。

彼が重荷じゃなく「嬉しい」と感じる尽くし方のタイミング、3章に書いてる。プロフのnoteから。

## ロング④｜何でもいいよ

### メイン
```
食べたいものも行きたい場所も、全部「何でもいいよ」「合わせるよ」って言ってる人。それ、優しさのつもりが彼を退屈させてるかも。
```

### ぶら下げ
```
「重いと思われたくない」「嫌われたくない」から、自分の意見を飲み込んで全部合わせる。すごく分かる。私もそうだった。

でも男から見ると、何も主張しない子って「一緒にいても何考えてるか分からない」「面白くない」になっちゃうんだよね。彼が求めてるのは「何でも合わせてくれる子」じゃなくて「一緒に楽しんでくれる子」。

「ここ行きたい!」「これ食べたい!」って言える子の方が、彼は一緒にいて楽しいし、手放したくなくなる。意見を言うのはわがままじゃなくて、あなたの魅力を見せること。

「重い」と思われずに自分を出す言い方、4章に例文付きで書いてるよ。プロフのnoteから。
```

```

正解は、小さなお願いをして、彼に「やってあげた」を積ませること。これだけで彼の中であなたへの気持ちが育ち始める。

## ロング⑥｜好き確認

### メイン
```
「私のこと好き?」って聞きたいのに、めんどくさいと思われそうで聞けなくて、一人でモヤモヤしてる人。その確認、聞いても不安は消えないよ。
```

### ぶら下げ
```
不安だから確認したい、すごく分かる。でも「好き?」って聞いて「好きだよ」って返ってきても、安心するのは一瞬で、次の日にはまた不安になる。確認って、不安の応急処置にしかならないんだよね。

しかも男から見ると、繰り返される確認は「安心させ続けないといけない相手」になって、だんだん重く感じる。

本当に必要なのは、彼に「好き?」って聞かせる側に回ること。あなたが余裕を持った瞬間、彼の方が「俺のこと好き?」って不安になる。立場が逆転するの。

彼を「確認する側」に回らせる関わり方、2章に書いてるよ。プロフのnoteから。
